# `11ty-starter`

`11ty-starter` is my personal starter project for building websites with 11ty. It's highly opinionated and is intended only for my personal use, so **use at your own risk.**

# `11ty-starter-common`

`11ty-starter-common` is a repo of tools that can be updated and contributed to. It's really the base of `11ty-starter`. It is a `git subtree`. See `package.json::scripts` where you will find `push-tools` (contributing commits upstream) and `pull-tools` (updating tools).
